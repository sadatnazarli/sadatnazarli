## What I'm building

### Verifex

**Verifex** is an API-first sanctions screening infrastructure product for modern compliance teams.

It helps companies screen individuals and entities against global sanctions, PEP, and risk sources with a focus on:

- high-recall sanctions detection
- lower false-positive review burden
- transliteration and cross-cultural name handling
- common-name false-positive controls
- source-policy-aware decisions
- fast API response times
- explainable match evidence
- transparent benchmarking

Verifex is designed around a simple principle:

> Compliance screening should not be a black box.  
> Every decision should come with evidence, uncertainty, and a clear reason.

Website: [verifex.dev](https://verifex.dev)
